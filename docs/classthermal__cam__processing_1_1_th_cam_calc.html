cam_obj )

Determines the centroid of heat for a given camera object.

This code takes a camera object as defined by mlx_cam.py and uses a weighted average of pixels to determine the centroid of in a single axis. mlx_cam.get_csv is used to return comma separated heat values ranging from 0-100, and those values are filtered to return only hotspots. Said hotspots are then used for the centroid calculation. This function returns a pixel location of the centroid ranging from 0-(self.NUM_PIXELS)

Parameters
cam_objA thermal camera object from mlx_cam.py
