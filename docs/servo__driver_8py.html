Detailed Description

Contains a class to drive basic servo motors.

Supports servo angle control via PWM pulse width. Initialization supports setting of upper and lower bounds for the pulse width determining servo angle. Further information about basic servo theory is available here: https://deepbluembedded.com/stm32-servo-motor-control-with-pwm-servo-library-examples-code/
